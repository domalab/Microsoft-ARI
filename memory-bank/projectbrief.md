# Project Brief: Azure Resource Inventory (ARI)

## Project Overview
Azure Resource Inventory (ARI) is a comprehensive PowerShell module designed to generate detailed Excel reports of any Azure environment. It provides Cloud Administrators and technical professionals with an easy and fast way to document their Azure environments.

## Core Requirements
- Generate complete resource documentation for Azure environments
- Create interactive Excel reports with resources organized by type
- Generate visual network diagrams of Azure environments
- Provide security analysis through Azure Security Center integration
- Support cross-platform usage (Windows, Linux, Mac, Azure Cloud Shell)
- Enable automation through Azure Automation Accounts
- Maintain low-impact with read-only operations

## Key Features
- **Complete Resource Documentation**: Detailed inventory of all Azure resources
- **Interactive Excel Reports**: Well-formatted spreadsheets with resources organized by type
- **Visual Network Diagrams**: Generate interactive topology maps of Azure environments
- **Security Analysis**: Integration with Azure Security Center (optional)
- **Cross-Platform Support**: Works on Windows, Linux, Mac, and Azure Cloud Shell
- **Automation-Ready**: Can be deployed via Azure Automation Accounts
- **Low-Impact**: Read-only operations with no changes to the environment

## Technical Stack
- PowerShell (7.0+ recommended, 5.1 supported)
- Azure PowerShell modules
- Excel report generation
- Azure REST API integration

## Project Status
Current version: v3.5
Key enhancements in v3.5:
- PowerShell Module Installation
- Full Automation Account Integration
- Azure REST API Integration (Support Tickets, Health Incidents, Advisor Score, Reservation Recommendations)

## Project Goals
- Provide comprehensive documentation of Azure environments
- Enable easy reporting and visualization of Azure resources
- Support automation and integration with existing workflows
- Maintain cross-platform compatibility
- Ensure low-impact operations with read-only access
