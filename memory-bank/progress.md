# Progress: Azure Resource Inventory (ARI)

## What Works

### Core Functionality
- PowerShell module structure and installation via PowerShell Gallery
- Azure resource discovery and inventory collection
- Excel report generation with resource organization by type
- Network diagram generation capabilities
- Cross-platform support (Windows, Linux, Mac, Azure Cloud Shell)
- Automation Account integration

### Recent Enhancements (v3.5)
- PowerShell Module Installation via standard PowerShell Gallery
- Full Automation Account Integration
- Azure REST API Integration for additional data sources:
  - Azure Support Tickets
  - Azure Health Incidents
  - Azure Advisor Score Data
  - Reservation Recommendations
